---

## 1. 검증 방법

### 백테스트 설계

- **대상 데이터**: IBTrACS WP 1951~2024 중 STY/TY 등급을 포함한 태풍 80개 무작위 선정
- **시작 조건**: 위도 5~25°N, 경도 110~175°E에서 발생한 태풍 (서태평양 전형적 발생 구역)
- **평가 지표**: 예측 경로와 실제 경로 간 Haversine 직선거리(km)
- **비교 기준**: 기상청 공식 태풍 예보 오차 기준 (+24h ~100km / +48h ~200km / +72h ~300km)
- **기저 모델(Baseline)**: Persistence — 시작점의 이동 방향과 속도를 그대로 유지한다고 가정

## 3. 기법별 상세 분석

### ① 유사 태풍 블렌딩 (Analog Method) — 가장 신뢰도 높음 ⭐

**원리**

실제로 발생했던 태풍 중 현재 조건(위치·기압·계절)과 가장 유사한 태풍 10개를 탐색하여, 그들의 이동 벡터를 유사도 가중 평균으로 합산합니다.

```
유사도 점수 = 1 / (1 + 거리/300km + 기압차/20hPa + 계절차×0.5)

가중치:  위치 유사도 50%  +  기압 유사도 30%  +  계절 유사도 20%
탐색 반경: 출발점 기준 800km 이내
선택 수: 상위 10개 태풍
```

**강점**

- 실제 관측 데이터를 직접 활용하므로 물리적으로 불가능한 경로가 나오지 않음
- 전향점(북서진→북동진), 계절별 경향 등이 자연스럽게 반영됨
- +72h 기준 319km로 기상청 기준(300km)에 근접

**한계**

- 과거에 유사한 사례가 없는 이례적 태풍에서 정확도 하락
- 데이터가 없는 시기(1950년대 이전)나 특수 지역 태풍에 취약

---

### ② LSTM Seq2Seq + Bahdanau Attention — 잠재적 최고 성능 🧠

**원리**

과거 24시간(4스텝)의 태풍 이동 시계열을 인코더에 입력하고, 어텐션 메커니즘이 어느 시점의 패턴에 집중할지 학습하여 최대 240시간(40스텝) 경로를 자동 회귀 방식으로 예측합니다.

```
입력 피처 (8개): dlat, dlng, dpres, lat_norm, lng_norm, pres_norm, sin_month, cos_month
아키텍처: Encoder(LSTM 2층) → Bahdanau Attention → Decoder(자동회귀)
앙상블: hidden_size = 48, 64, 96 세 모델 평균
훈련: L1Loss / Adam(lr=0.001) / CosineAnnealingLR / Early Stopping(patience=12)
훈련 데이터: IBTrACS 슬라이딩 윈도우 시퀀스
```

**강점**

- 과거 24h 시계열을 보기 때문에 단순 관성보다 방향 전환 패턴 학습 가능
- 앙상블 평균으로 단일 모델의 극단적 예측 완화
- 충분히 훈련되면 유사 태풍 블렌딩 수준 또는 그 이상 가능

**현재 상태 및 한계**

- `lstm_model_h48.pt`만 훈련 완료 (h64, h96 미완료) → 아직 단일 모델로만 동작
- 훈련 데이터가 6시간 간격 이동량(dlat, dlng)이므로 갑작스러운 방향 전환에 취약
- 앙상블 3개 모두 완료 후 재검증 필요

---

### ③ GradientBoosting ML — 단일 스텝 정확도 준수 🤖

**원리**

11개 기상 피처를 입력으로 다음 6시간의 이동량(dlat, dlng, dpres)을 각각 독립적으로 예측합니다.

```
입력 피처 (11개):
  lat, lng, pressure, wind_1min_ms, wind_10min_ms, diameter_km,
  month, prev_dlat, prev_dlng, lat×sin(월), lat×cos(월)

모델: GradientBoostingRegressor (n_estimators=300, max_depth=5, lr=0.05)
훈련 샘플: 191,323개 (테스트 47,831개)
```

**강점**

- 6시간 단일 스텝 위치오차 15.75km로 테이블형 데이터에 강함
- 과적합 억제 구조 (subsample=0.8, min_samples_leaf=10)
- `prev_dlat`, `prev_dlng` 피처로 관성 패턴 포착

**한계**

- Persistence(15.9km) 대비 스킬 스코어 +0.9% — 사실상 유사한 수준
- 1스텝 오차가 누적되면 20~40스텝에서 급격한 오차 증가
- 각 스텝이 독립적으로 예측되어 경로의 연속성 보장이 어려움

---

### ④ 물리 모델 — 항상 작동하는 최후 폴백 ⚙️

**원리**

위도, 기압, 해수면 온도(SST)를 기반으로 태풍의 이동 방향과 강도를 물리 공식으로 계산합니다.

```
저위도 (< 20°N): 북서진 (Beta drift 지배)
중위도 (20~28°N): 전향 구간 (북서 → 북북서)
고위도 (> 28°N): 북동진 (편서풍 지배)

기압 생애주기:
  0~24h : 강화 단계 (SST ≥ 28°C 조건)
  24~72h: 절정 유지
  72h+  : 위도 기반 약화 가속 → 소멸 (pres ≥ 1008hPa)
```

**강점**

- 언제나 결과를 반환 (데이터/모델 불필요)
- 전형적인 태풍 생애주기(STY→TY→TS→TD→소멸)를 재현

**한계**

- +24h 오차 332km — 기상청의 3배 이상
- 계절, 지형, 순간적 기압 변화 등을 반영하지 못함
- 상층 기류나 다른 태풍과의 상호작용 미반영

## 4. 종합 신뢰도 평가

```
신뢰도 순위:

  1위 ⭐ 유사 태풍 블렌딩   → 기상청 기준의 1.0~1.5배  (교육적 활용 충분)
  2위 🧠 LSTM (훈련 완료 시) → 잠재적으로 1위와 동등 이상
  3위 🤖 GBM ML             → 단기(6h)는 준수, 장기 누적 오차 높음
  4위 ⚙️ 물리 모델          → 기상청 기준의 3배 이상 (방향성 참고 수준)
```

| 활용 목적 | 권장 여부 |
|-----------|-----------|
| 교육·체험용 시각화 | ✅ 충분 |
| 태풍 경로 경향 파악 | ✅ 유사 태풍 블렌딩 한정 |
| 실제 기상 예보 보조 | ⚠️ 부족 (앙상블 LSTM 완성 후 재평가 필요) |
| 공식 재난 대응 활용 | ❌ 부적합 (기상청 공식 예보 사용 권장) |

## 5. 개선 방향

| 우선순위 | 항목 | 예상 효과 |
|----------|------|-----------|
| 🔴 High | LSTM 앙상블 3개 훈련 완료 (h64, h96) | 경로 곡률 예측 개선 |
| 🔴 High | 유사 태풍 블렌딩을 1순위 폴백으로 변경 | 즉각적 정확도 향상 |
| 🟡 Medium | GBM 입력에 이전 3~4스텝 시계열 피처 추가 | 장기 누적 오차 감소 |
| 🟡 Medium | 검증 데이터셋 확대 (200개+, 10개년 홀드아웃) | 신뢰도 평가 강화 |
| 🟢 Low | 앙상블 가중치 동적 조정 (SST/위도별 모델 선택) | 특수 케이스 정확도 개선 |

---

## 6. 결론

TyphoonPath는 **체험·교육 목적의 태풍 경로 시각화 서비스**로서 충분한 신뢰도를 갖추고 있습니다.

특히 유사 태풍 블렌딩 기법은 기상청 공식 예보의 1~1.5배 수준 오차로 실제 태풍 경향을 상당히 정확하게 반영하며, 강도 분류 정확도 78.8%는 교육 시각화 목적으로 충분한 수치입니다.

단, 실제 기상 예보나 재난 대응 보조 도구로 활용하기 위해서는 LSTM 앙상블 완성 및 추가 검증이 필요합니다.
