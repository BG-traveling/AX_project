## 4. 기능 요구사항 (Functional Requirements)

### FR-01: 태풍 데이터 조회
- 기상청 기상자료개방포털 또는 공공데이터포털 API 연동
- 1951~2024년 태풍 목록 및 경로(위경도, 기압, 풍속, 날짜) 제공
- Redis 캐싱으로 반복 요청 최적화

### FR-02: 지도 시각화
- Leaflet.js 기반 인터랙티브 지도
- 태풍 경로를 폴리라인으로 표시
- 각 관측점을 마커로 표시 (기압/풍속 정보 팝업)

### FR-03: 변수 조작 UI
- 기압 범위: 850hPa ~ 1010hPa (슬라이더)
- 온도 범위: 관련 해수면 온도 (슬라이더)
- 조작 시 예상 경로 실시간 업데이트 (간단한 시뮬레이션 모델)

### FR-04: 경로 예측 제출
- Leaflet DrawControl 또는 커스텀 드로잉으로 경로 입력
- 위경도 좌표 배열로 백엔드 전송
- 제출 완료 시 비교 화면 전환

### FR-05: AI 피드백 생성
- FastAPI → Claude API 호출
- 입력 데이터: 사용자 예측 경로, 실제 경로, 사용자가 조작한 변수값
- 출력: 오차 분석, 원인 설명, 학습 포인트 (한국어)
